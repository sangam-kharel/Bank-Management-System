# 🏦 Bank Management System

A console-based Bank Management System written in C, built as part of the TU BCA First Semester C Programming course. The project simulates a real-world banking workflow with separate customer and admin interfaces, persistent binary file storage, and PIN-based authentication.

## ✨ Features

### 👤 Customer
- Create & delete account
- Login with PIN authentication
- Deposit, withdraw, and transfer money
- Check balance and view transaction history
- Update profile and change PIN

### 🛡 Administrator
- Secure admin login
- View all customer accounts
- Search and delete accounts
- View bank-wide statistics

## 🚀 Getting Started

### Prerequisites
- GCC compiler
- GNU Make
- Linux / macOS (or Windows with MinGW)

### Build & Run

**Clone the repository**
```bash
git clone https://github.com/your-username/Bank-Management-System.git
cd Bank-Management-System
```

**Build**
```bash
make
```

**Run**
```bash
./bank
```

**Clean build files**
```bash
make clean
```
